Donut or not!


Simple exercise to learn how to push a deep learning model to the cloud and run inference in a serverless fashion. This app is built using FastAPI, Docker, AWS Lambda, SAM. The model is built using FastAI on top of PyTorch using a resnet32 base model.
Click on the button below to upload some jpg and classify it between the 3 known classes - Donut | Bagel | Vada.

Only images of type JPG are supported
